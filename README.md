# Grok Voice Assistant Repo

This repo is for the Grok CLI assistant.


## Features

- Download research papers from arXiv.
- Extract text from PDFs.
- Convert text to audio.
- Play audio files.

## Setup

1. Install dependencies: wget, pdftotext, vlc, python venv.

2. Run the assistant.

## Usage

Ask the assistant to play a paper on robotic manipulators.
